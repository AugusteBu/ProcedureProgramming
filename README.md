# C Provedure Programming 

This is my 1st year University work of C programming language. Collection of code for learning how to write programs, manage computer memory, learn algorithms and basic syntax for programming languages. It has basic read, write operations, error handling, assertions etc.

Created by: Augustė Buklerytė

## Folder Directory and Skills Progress

Folder order goes by weeks

| Folder Name | What the Code Does |
| :--- | :--- |
| **01-hello-world** | Hello world and Code Basics |
| **week2** | Base Conversion calculations, Filtering, Equations |
| **03-binary-database-filter** | Filtering data with generated data |
| **04-doubly-linked-list** | Doubly Linked List implementation |
| **04-math-and-sequence-algorithms**| Math solver for Quadratic, Fibonacci, GCD, LCM, Max, Counter. |
| **05-arrays** | Array management, saving, editing, generating data |
| **06-string_parsing_and_validation**| Text, counts letters, and edits strings with different functions |
| **07-cli_utilities** | Cli utilities, error handling, generative data, calculations |
| **08-pointers_and_memory_management** | Memory Allocations and handling them |
| **09-data_structures_and_structs**| Custom data holders, like a Stack |
| **10-assertions** | Assertions to automatically tests code to make sure it has no errors |
| **week11** | Assertions to automatically tests code to make sure it has no errors |
